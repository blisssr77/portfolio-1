    Project One

    WeLike - Social Media Application

    I developed a dynamic full-stack social media platform with a modern, responsive design using React.js, TypeScript, and Tailwind CSS, delivering a mobile-app-like user experience. The platform features secure Appwrite-powered authentication, drag-and-drop post creation, editing, and profile customization. Key functionalities include an explore page for content discovery, options to like and save posts, and detailed post and profile views. Efficient data fetching with React Query enhances performance, while backend services managed by Appwrite ensure scalability, security, and smooth application management.

    Project Two

    GetConnected - Finding Learner/Freelancer Application

    I developed a full-stack web application connecting students with freelancers, featuring profile creation, freelancer search, and learning management tools. The responsive frontend, built with React and Tailwind CSS, provides a user-friendly experience. The backend, developed with Node.js, Express, and MongoDB (using Mongoose), includes JWT-based authentication for secure access. Designed for scalability and performance, this platform offers a reliable solution for connecting users and supporting educational engagement.

    Project Three

    Briize - Fragrance Blog

    I developed a full-stack web application for blog management, allowing users to create, edit, delete, and view posts. The application includes a responsive, interactive frontend built with JavaScript, CSS, and EJS for seamless user experiences and real-time updates. The backend, built with Node.js, Express, and MongoDB (using Mongoose), efficiently manages data storage and retrieval, ensuring scalability, security, and a smooth blogging experience.
